# Журнал решений

Статус: 🟡 Design draft

Этот файл — облегчённый индекс решений по внедрению.

Используйте его, когда в организации ещё нет формального архитектурного совета, но проект всё равно нуждается в видимом следе принятых решений.

---

## Как использовать

1. Каждое значимое решение получает номер ADR.
2. Краткое резюме решения фиксируется здесь.
3. Полный документ хранится в `examples/decisions/adr-XXX-*.md` или в репозитории проекта.
4. Журнал решений проверяется на каждом phase gate.
